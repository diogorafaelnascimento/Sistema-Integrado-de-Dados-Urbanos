# Sistema Integrado de Dados Urbanos
## Urban Data Integrated System
Python | CLI Application | Data Processing | Urban Analytics

Ferramenta desenvolvida em Python para coleta, processamento e análise estruturada de dados urbanos.

---

## Descrição

O projeto consiste em um sistema interativo em linha de comando que permite:

- Inserção estruturada de dados urbanos
- Processamento de múltiplos lotes
- Cálculo automático de indicadores
- Classificação baseada em predominância estatística
- Geração de diagnóstico sintético

O sistema utiliza modelagem por dicionários, validação de entradas e regras condicionais.

---

## Funcionalidades

### Coleta de dados

- Dimensões da via
- Tipo de pavimentação
- Infraestrutura urbana
- Tipo de ocupação dos lotes
- Área construída
- Área permeável
- Número de moradores

### Processamento

- Índice de Aproveitamento (CA)
- Taxa de Ocupação (TO)
- Índice de Permeabilidade (IP)
- Densidade populacional líquida (hab/km²)
- Classificação morfológica automática
